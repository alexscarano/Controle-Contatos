# Controle de Contatos

Uma aplicação web desenvolvida com o objetivo de aprender novas tecnologias.

## Tecnologias utilizadas

- **C#** (ASP.NET)
- **HTML**
- **CSS**
- **JavaScript**

## Sobre

Este projeto permite gerenciar contatos de forma simples e intuitiva.  
Ideal para praticar conceitos de back-end e front-end no desenvolvimento web.

## Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/alexscarano/Controle-Contatos.git
   ```
2. Abra e execute o projeto no Visual Studio.
